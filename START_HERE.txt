### 방법 1️⃣: Windows 사용자 (가장 간단) ⭐ RECOMMENDED

1. 이 폴더에서 마우스 우클릭
2. "PowerShell 여기서 열기" 또는 
   "명령 프롬프트 여기서 열기" 선택
3. 다음 명령 실행:
   
   .\deploy.bat
   
4. 메뉴에서 선택:
   1) Git 저장소 초기화
   2) GitHub에 업로드
   3) GitHub Pages 설정 안내

### 방법 2️⃣: 수동 배포 (모든 OS)

1. GitHub 저장소 생성 (QUICKSTART.md 참고)
2. PowerShell/Terminal에서 명령 실행
3. 자세한 방법은 QUICKSTART.md 파일 참고

### 방법 3️⃣: 기술자 전용 (고급)

자세한 설정은 DEPLOYMENT.md 참고
